Welcome to Ener-G.

Ener-G is a web app that helps you track your energy consumption patterns. Its simple! You provide your data. We display your data in beautiful charts. Now, you can easily understand your energy consuption patterns. Click here to start.

Try It

Go to MyEnergy page. Enter your data, using the info from your energy bill or upload an .xsl file, and let our app create comprehensive charts.

Get the Code

The source code for this application is maintained in this GitHub repository. The repository includes several adapters to use the application with different data persistence solutions: in-memory store, REST API, REST with JSONP, Node.js, MongoDB, etc.
